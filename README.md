## Requirements

Before starting üèÅ, you need to have [Git](https://git-scm.com), [Node](https://nodejs.org/en/) and [Yarn](https://yarnpkg.com) installed.

## Starting

```bash
# Clone this project
$ git clone git@github.com:Thiago-spart/bank-dashboard.git

# Access
$ cd bank-dashboard

# Install dependencies
$ yarn

# Run the development project version
$ yarn dev

# The server will initialize in the <http://localhost:3000>
```
